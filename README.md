# Replicação da Tela do Instagram - Projeto de Design

Este é um projeto de design criado para demonstrar minhas habilidades no uso do Figma. 
A proposta é replicar a tela inicial do Instagram, focando muito na estética, layout e na experiência visual. 

## Tasks

- **Interface Principal**: Replicação da tela inicial do Instagram, incluindo feed de posts, stories, e navegação.
- **Design Responsivo**: Adaptação de layouts para diferentes tamanhos de tela (mobile, tablet, desktop).
- **Elementos de UI**: Criação e customização de botões, ícones, menus e outros componentes interativos.

## Organização do Repositório

O repositório está estruturado da seguinte maneira:

- `assets/`: Contém imagens, ícones e outros recursos gráficos usados no projeto.
- `figma/`: Arquivo principal do Figma com o design da tela replicada.
- `resources/`: Pastas com materiais de apoio, como fontes e paletas de cores utilizadas.

## Tecnologias e Ferramentas Utilizadas

- **Figma**: Para criação do design.
- **Adobe Photoshop / Illustrator** (se necessário para edição de imagens).
- **Sketch ou outras ferramentas** (se aplicável).

## Como Visualizar o Projeto

Para visualizar o design, você pode abrir o arquivo do Figma diretamente ou ver as imagens exportadas, que estão disponíveis na pasta `assets/`.

1. **Abrir o Figma**: [Link para o Figma](https://www.figma.com) (Se tiver um link público ou compartilhado para seu arquivo Figma).
2. **Visualizar as Imagens Exportadas**: Acesse as pastas `assets/` para ver as imagens.
